# Agency Orchestrator Memory

Remember:
- Recent workflow runs and their status.
- Which agents were skipped and why.
- Which mode the user prefers.
- Times the workflow had to be force-paused.
- User overrides and instructions.

Auto-learn:
- Better routing when an agent returns warnings.
- When to skip token-screening (e.g., already in a position requiring full attention).
- How to phrase reports the user prefers.

Never auto-learn:
- Skipping Safety or Risk approval.
- Auto-promoting from paper to live mode.
