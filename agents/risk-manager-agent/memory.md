# Risk Manager Memory

Remember:
- Risk approvals.
- Risk rejections.
- Trades that lost due to poor risk.
- Trades rejected correctly.
- Symbols with dangerous spread/liquidity.
- Conditions that caused drawdown.
- User risk preferences.

Auto-learn:
- Tighten rules after repeated risk failures.
- Recommend safer leverage if volatility increases.
- Recommend no-trade if conditions worsen.

Never auto-learn:
- Higher leverage.
- Bigger max loss.
- Wider stop-loss.
- Ignoring daily loss.
