# Execution Memory

Remember:
- Order IDs.
- Fill prices.
- Slippage.
- Failed orders.
- Partial fills.
- Symbol precision issues.
- API errors.
- Stop placement failures.
- Actual vs expected fees.

Auto-learn:
- Which symbols have bad fills.
- Which order type works better per symbol.
- Which times have high slippage.

Never store:
- API keys.
- Secret keys.
- Private credentials.
