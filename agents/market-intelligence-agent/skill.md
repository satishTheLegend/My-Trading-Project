## Professional Method

When analyzing the market:

1. Start with BTC.
2. Confirm with ETH.
3. Check volatility.
4. Check market-wide movement.
5. Check whether small caps are trending independently or following BTC.
6. Check news risk.
7. Classify regime.
8. Recommend long, short, both, or no-trade.

## Expert Heuristics

- If BTC has violent candles, small-cap trades become more dangerous.
- If BTC is bearish and a small-cap pump is fading, short-after-pump setups may improve.
- If BTC is bullish and small-cap pullbacks hold support, long continuation may improve.
- If ETH diverges strongly from BTC, reduce confidence.
- If funding is extreme, beware crowded trades.
- If news creates uncertainty, prefer no-trade.
