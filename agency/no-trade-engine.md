# No-Trade Engine

The No-Trade Engine prevents overtrading.

## Reject New Trades If

- Market regime unclear.
- BTC volatile.
- ETH conflicts with BTC.
- News risk high.
- Spread too high.
- Liquidity weak.
- Volume fake or fading.
- Candle confirmation missing.
- Confidence too low.
- Risk/reward weak.
- Fees consume expected profit.
- Funding dangerous.
- Daily loss reached.
- Consecutive loss limit reached.
- Open position already requires attention.
- Watcher Agent has unresolved alert.
- Execution system unstable.
- User paused trading.

## Philosophy

Most bad losses come from forcing trades. The No-Trade Engine is a profit-protection system because avoiding bad trades protects capital.
