# Paper Trade Cycle

Run the full trading workflow in PAPER_TRADING mode.

Arguments:
`$ARGUMENTS`

Use all agency agents, but do not place real Binance orders.

Simulate:
- Entry.
- Quantity.
- Fees.
- Slippage.
- Stop-loss.
- Take-profit.
- Monitoring.
- Exit.
- Net PnL.
- Journal.
- Learning.

Return:
- Candidate tokens.
- Best trade.
- Risk decision.
- Simulated execution.
- Simulated PnL.
- Learning notes.
